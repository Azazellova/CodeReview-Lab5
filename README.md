## Основные изменения

### 1. Меню выбора

* Выведено информативное меню вместо набора чисел.

### 2. Класс NumSum

* Класс был разбит на отдельные классы и интерфейсы.

### 3. Класс `Line`

* Копирование координат при изменении начала и конца линии, чтобы избежать ссылочной зависимости.

### 4. Классы `Point`

* Название классов связанных с точками исправлено на соответствующее Google Code Style.

### 5. Класс `Validator`

* Добавлены методы для безопасного ввода строки.
* Изменен принцип проверки введенных данных в методах `inputInteger`,`inputPositiveInteger`
* Удалены неиспользуемые методы

### 6. Общие улучшения

* Добавлена XML-доукментация
* Исправлены ошибки, из-за которых задания 3.1 и 5.6 некорректно работали с пользовательским вводом и копированием данных.
